## Setup Instructions

### 1. Add Project Images
Save your project images in the `images` folder with these names:
- `tower-segments.jpg` - Steel lattice tower segments prepared for assembly
- `mobile-crane.jpg` - 80-ton mobile crane for tower lifting operations
- `tower-installation.jpg` - Tower installation using crane and rigging equipment
- `safety-team.jpg` - Safety team conducting site inspection

### 2. View the Website
Simply open `index.html` in your web browser.

### 3. Deploy to nareshandco.com

#### Option A: GoDaddy Hosting
1. Purchase domain at godaddy.com
2. Sign up for GoDaddy hosting
3. Upload files via File Manager or FTP:
   - index.html
   - styles.css
   - script.js
   - images folder (with all images)
4. Point domain to hosting

#### Option B: Netlify (Recommended - Free & Easy)
1. Create account at netlify.com
2. Drag and drop your project folder
3. Get free subdomain (e.g., nareshandco.netlify.app)
4. Purchase domain separately and connect it in Netlify settings

#### Option C: Vercel (Free & Fast)
1. Create account at vercel.com
2. Import your project
3. Deploy with one click
4. Connect your domain in settings
